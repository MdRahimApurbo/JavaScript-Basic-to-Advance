# If Else Statement

**This else-if statement**

```javascript
var age = 18;
var gender = 'male';

if (age >= 18 && gender == 'male') {
    document.write('You are Adult Boy');
} else if (age >= 18 && gender == 'female') {
    document.write('You are Adult Girl');
} else {
    document.write('You are Child Boy');
}

```
